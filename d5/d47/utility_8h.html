Exports a table to a file in a formatted manner.

This function writes a table to a file, appending rows formatted by a user-provided function. Each row includes an index and the formatted content of the table element.

Parameters
filenameThe name of the file to write to.
tableA pointer to the table data in memory.
table_sizeThe number of elements in the table.
element_sizeThe size (in bytes) of a single table element.
format_rowA function pointer that formats a single table element into a string. The returned string must be dynamically allocated and will be freed after use.
begin_markerA marker string written at the beginning of the table in the file.
end_markerA marker string written at the end of the table in the file.
  • Opens the file in append mode and writes the begin_marker at the start.
  • Iterates through the table elements, formats each row using the format_row function, and writes it to the file.
  • Writes the end_marker at the end and closes the file.
Note
If the file cannot be opened, the program terminates with an error.
Warning
The format_row function must return dynamically allocated strings, as they will be freed internally.

int get_array_size ( int  index_declaration_array)

Calculates the total size of an array based on its dimensions.

Parameters
index_declaration_arrayThe declaration index of the array.
Returns
The total number of elements in the array.
  • Iterates through all dimensions to calculate the product of their sizes.
  • Checks that the lower bound of each dimension is not greater than the upper bound.
Note
If a lower bound exceeds its upper bound, an error message is printed, and the program terminates.
